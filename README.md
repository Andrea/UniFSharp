UniFSharp - F# Build Tools for Unity
===

UniFSharpとは?
---
UniFSharpはUnity(Windows用)のEditor拡張です。
Unity Editorの"Assets/Create/F# Script/NewBehaviourScript" のメニューから選択するだけで、F# Scriptを作成できるようになります。
そして、Unity Editor上でF# ScriptをDLLにビルドすることができます(MSBuild利用)。
F#(関数型プログラミング言語)でUnityのゲーム開発がしやすくなります。
また、オープンソースキャラクターのユニティちゃん(ユニティ・テクノロジーズ・ジャパン提供)をマスコットキャラクターに採用。
ユニティちゃんの音声で時報やイベント、ビルド結果の通知する機能を提供しています。
